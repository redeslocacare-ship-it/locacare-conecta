# LocaCare

Aplicação web para captação de leads (site público) e gestão de locações (área administrativa).

## Rodando localmente

```sh
npm i
npm run dev
```

## Estrutura

- `src/pages/Index.tsx` — Site público (conversão)
- `src/pages/Auth.tsx` — Login/Cadastro
- `src/pages/admin/` — Área administrativa
- `docs/` — Documentação técnica (arquitetura, modelo de dados, fluxos)
